ctrl + a 行首
ctrl + e 行尾
Ctrl + u：从当前光标所在位置向左剪切全部命令 
Ctrl + k：从当前光标所在位置向右剪切全部命令
alt + f:光标向后移动一个单词
alt + b:光标向后移动一个单词
ctrl + d：删除当前光标的字符
ctrl+t 交换当前字符和前一个字符的位置
alt+t 交换当前单词和前一个单词的位置
alt+u 把光标之后的单词变成大写
alt+l 把光标之前的单词变成小写
alt+c 把当前单词变成首字母大写的单词
ctrl+w 剪切光标前的单词
